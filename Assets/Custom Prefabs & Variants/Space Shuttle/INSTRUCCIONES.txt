PARA HACER PREFABS DE MODULES / CORRIDORS NUEVOS
	YA SEA PARA CAMBIAR EL COLOR O LA FORMA
		POR EJEMPLO UNA SECCION NUEVA QUE TENGA COLORES DISTINTOS
		Y UNA ARQUITECTURA CARACTERISTICA COMO LA PUEDE TENER UN PUZZLE

OPCIONAL / RECOMENDADO
	COGED VARIANTES DE LA CARPETA "1 VARIANTS"
		SI ALLI NO ESTAN LOS QUE QUEREIS
			HACED PREFAB VARIANTS DE LOS ASSETS ORIGINALES

OBLIGADO
	CREAD UN PREFAB NUEVO PARA EL MODULE / CORRIDOR QUE QUERAIS HACER
		EL COLOR SE PUEDE CAMBIAR TANTO EN LA VARIANTE COMO EN EL MODULE / CORRIDOR
			COMO PONE ARRIBA ES RECOMENDADO HACERLO DESDE LA VARIANTE PARA HACER CAMBIOS DE COLOR EN MASA
				SI LOS CAMBIAIS DESDE EL MODULE / CORRIDOR UNITY DEJARA DE ACTUALIZARLOS SI LA VARIANTE ROOT CAMBIA

		SEA COMO SEA NO ELIMINEIS / MODIFIQUEIS LOS MODULE / CORRIDOR ANTERIORES
			MUY IMPORTANTE
				PUES CAMBIARA LOS NIVELES CONSTRUIDOS HASTA AHORA

	UNA MANERA MUY RAPIDA DE HAERLO ES COGIENDO UN PREFAB YA HECHO Y DUPLICARLO ( CTRL + D )
		DESPUES LO EDITAIS

	TAMBIEN SE PUEDE LLEVAR A UNA ESCENA Y HACER UNPACK PARA DESPUES VOLVER A HACERLO PREFAB
